# 卖家 NPC 提示框朝向设计

## 目标

让卖家 NPC 头顶的提示框保持世界空间位置并跟随 NPC，同时始终朝向当前摄像机，类似参考图中的 UI 提示框。

## 方案

新增通用 `CameraFacingUI` 组件，挂载到卖家 NPC 的 `emoji` 提示节点。组件在运行时查找主摄像机，并将提示节点的世界旋转同步为摄像机世界旋转；不修改 NPC 本体旋转、位置或移动逻辑。若摄像机尚未就绪，则保留当前旋转并在后续帧继续尝试。

## 范围

- 新增：`assets/_Scripts/CameraFacingUI.ts`。
- 在 `DevScene` 中为卖家 NPC 提示节点挂载该组件。
- 不改提示框显示/隐藏条件、NPC 队列、售卖库存、玩家或镜头跟随逻辑。

## 验证

运行场景并从不同镜头角度观察卖家 NPC：提示框应始终正对摄像机，NPC 移动时提示框跟随；旋转镜头不会使提示框侧翻或背对玩家。
